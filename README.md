# Replace Deprecated Library Function JavaScript

This example demonstrates how to transition away from a hypothetical deprecated library function by replacing it with a native JavaScript implementation. It showcases a simple data transformation task, illustrating how developers can evaluate alternatives and update their codebase when a dependency reaches end-of-life.

## Language

`html`

## How to Run

Save the code as `index.html`. Open `index.html` in any web browser.
